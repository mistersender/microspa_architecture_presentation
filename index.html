Let's Talk Vernacular

SPA

Single-Page Application

(Typically uses a JS framework)

Monolith

Single, large code base.

* a single large code base (or potentially 2-3 code bases for MVCs etc) that control all of the code for a site (or sites). * typically have a lot of shared components * once a monolith is learned, it's typically super fast to code in

Micro SPA

  • Code base broken out into separate, distinct pieces
  • Performs a single task
  • Intended to work as part of a whole website, but can stand alone.

Comparing MicroSPAs to Monoliths

Using a generic E-Commerce site

Monolith Repos:

  • widgets_galore

MicroSPA Repos:

  • widgets_galore_portal
  • home_page
  • sales_landing_page
  • search
  • checkout
  • my_account
  • update_user_info

can reuse my_account etc on other portals
